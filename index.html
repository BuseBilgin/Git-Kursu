Windows Git Kurulumu Windows için git yükleyicisini indirin aaa Resmi git sitesinden https://git-scm.com/downloads Windows seçeniğini seçerek indiriyoruz. Yükleyiciyi başlatın ve cihazınızda değişlik yapmasını izin verin. Git kurulumunu tamamlayın Uygulamanın yüklenmesini istediğiniz yeri seçin (varsayılan yere yüklenmesi önerilir). Yükleyici, başlat menüsünde klasörü oluşturmayı önerecektir. Next'e tıklamanız yeterlidir. Bir bileşen seçim ekranı görünecektir. Özel olarak değiştirmeniz gerekmiyorsa varsayılanları bırakın ve Next'e tıklayın. Git ile kullanmak istediğiniz bir metin düzenleyici seçin. Notepad++'ı (veya tercih ettiğiniz herhangi bir metin düzenleyiciyi). Burada varsayılan Branch ismini istediğiniz gibi seçebilir veya git'in önerdiğini kullanabilirsiniz Bu kurulum adımı, PATH ortamını değiştirmenize izin verir. PATH, komut satırından bir komut çalıştırdığınızda dahil edilen varsayılan dizin kümesidir. Bunu ortadaki (önerilen) seçimde bırakın ve Next'e tıklayın.
